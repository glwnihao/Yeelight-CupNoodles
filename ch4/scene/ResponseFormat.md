# ResponseFormat

```
{
			    "payload": {
			        "commands": [
			            {
			                "ids": [
			                    "6-199"
			                ],
			                "status": "SUCCESS"
			            }
			        ]
			    },
			    "requestId": "6529877997918554205"
			}
```

| Params            | Type          | Value             | Remarks              |
| ----------------- | ------------- | ----------------- | -------------------- |
| commands → ids    | Array<String> |                   | Device ID collection |
| commands → status | String        | `SUCCESS / ERROR` | Perform result       |
|                   |               |                   |                      |
|                   |               |                   |                      |
